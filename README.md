## `architect help [COMMAND]`

display help for architect

```
USAGE
  $ architect help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v2.1.3/src/commands/help.ts)_

## `architect init`

Create an architect.json file for a service

```
USAGE
  $ architect init

OPTIONS
  -a, --author=author
  -d, --description=description
  -h, --help                     show CLI help
  -k, --keywords=keywords
  -l, --license=license          [default: MIT]
  -n, --name=name                [default: architect-cli]
  -v, --version=version          [default: 0.1.0]
```

_See code: [src/commands/init.ts](https://github.com/architect-team/architect-cli/blob/v0.1.6/src/commands/init.ts)_

## `architect install`

Install dependencies of the current service

```
USAGE
  $ architect install

OPTIONS
  -h, --help           show CLI help
  -p, --prefix=prefix  Path prefix indicating where the install command should execute from.
  -r, --recursive      Generate architect dependency files for all services in the dependency tree.
```

_See code: [src/commands/install.ts](https://github.com/architect-team/architect-cli/blob/v0.1.6/src/commands/install.ts)_

## `architect start`

Start the service locally

```
USAGE
  $ architect start

OPTIONS
  -c, --config_path=config_path  Path to a config file containing locations of each service in the application
  -h, --help                     show CLI help
```

_See code: [src/commands/start.ts](https://github.com/architect-team/architect-cli/blob/v0.1.6/src/commands/start.ts)_
